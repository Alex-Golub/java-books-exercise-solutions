> What expression produces the third-to-last digit of the number (the 100s place)?

> > 10s place => `number / 10 % 10`  
> > 100s place => `number / 100 % 10`

> **printMaxMin**
>
> > ```java
> > int max, min = 10;

> > max = max + 6;                 // max = 23

> > System.out.println(max + min); // 36
> > System.out.println(max);       // 23
